JSON Formatter

Paste any JSON into the input pane. With Auto enabled it validates and formats as you type; turn Auto off and a Format JSON → button appears so nothing runs until you click it.

Three output views: Format (pretty-printed, 4-space indent, collapsible tree — click the ▶/▼ arrows or use Expand All / Collapse All), Minify (single line, no whitespace), and Tree (an ASCII directory-style view of the key structure).

Invalid JSON shows the exact parser error instead of silently failing. Use Upload to load a .json file from disk, Copy to grab the formatted result, Download to save it back out, or Fullscreen to view the output full-page (with its own Copy/Download and a Minimize button, or press Esc).

History keeps your last 5 valid entries, saved to this browser's local storage whenever you click away from the input, format manually, or upload a file. Open the History dropdown and click any entry to reload it.

Text Compare

Paste two versions of a file into Original and Modified. The diff is computed line-by-line and rendered as Side by side (two columns) or Inline (one unified stream with +/− markers).

Turn on Ignore whitespace to treat lines that differ only in spacing as identical. Swap flips the two panes, and the +added / −removed counters update live above the output.

Privacy & how it runs

Every tool runs entirely in your browser. Nothing you paste, upload, or format is sent to a server — there's no backend involved, no account, and no tracking of file contents. Closing the tab clears everything.
